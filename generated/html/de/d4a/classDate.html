Detailed Description

Class representing a date (day, month, and year). Built-in date structures combined dates with time within the day, which can lead to accumulated errors when trying to increment dates in whole-day steps.

Since converting a date to a string is the most common operation, store the date in a way that allows easy extraction of the day, month, and year. Allow 5 bits for the day and 4 for the month. This also allows easy comparison.
