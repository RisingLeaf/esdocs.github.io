Detailed Description

File paths and file handling are different on each operating system. This class stores the path, on each operating system, to the game's resources - images, data files, etc. - and also to the "configuration" directory where saved games and other user-specific information can be stored. It also provides an interface for file operations so that the rest of the code can be completely platform-agnostic.
