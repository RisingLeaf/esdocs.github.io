Detailed Description

A class containing a list of text replacements. Text replacements consist of a key to search for and the text to replace it with. One key can have multiple potential replacement texts, with the specific text chosen being defined by whichever replacement is the last valid replacement for that key, with replacement validity being defined by a ConditionSet.
