Detailed Description

Class to be used for enforcing a certain frame rate. This ensures that the frames are not drawn faster than a certain speed, but if the calculations or the graphics cannot keep up it will allow things to go slower for a few frames without trying to "catch up" by making the subsequent frame faster.
