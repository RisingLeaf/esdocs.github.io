Detailed Description

Class for queuing up tasks that are to be executed in parallel by using every thread available on the executing CPU. The queue is also responsible to execute follow-up tasks that need to executed after the async task, for example uploading a loaded to the GPU (which needs to happen on the main thread on OpenGL).
