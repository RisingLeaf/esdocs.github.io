32// Object to hold a set of "stars" to be drawn as a backdrop. The star pattern
33// repeats every 4096 pixels. The pattern is generated by a random walk method
34// so that some parts will be much denser than others, which is visually more
35// interesting than if the stars were evenly spread out in perfectly random
36// noise. If the view is moving, the stars are elongated in a motion blur to
37// match the motion; otherwise they would seem to jitter around.
