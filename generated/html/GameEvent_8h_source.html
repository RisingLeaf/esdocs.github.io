36// This represents something that happens that changes the universe. For
37// example, a system may be taken over by a different government, or a new type
38// of ship or weapon may become available for purchase. Events that do not
39// specify a date on which they occur will happen in response to missions. An
40// event always sets the "event: <name>" condition when it occurs, which allows
41// you to use the mission framework to specify a message that can be shown to
