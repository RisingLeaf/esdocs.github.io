Detailed Description

A ShipEvent is anything of significance that one ship does to another. These events might change one government's attitude toward the player, or may result in a certain message or dialog being displayed.

Enumerator
NONE 

This is a "null" event.

ASSIST 

This ship did something good for the given ship.

SCAN_CARGO 

This ship scanned the given ship's cargo. This is not necessarily an act of aggression, but it implies mistrust or underhanded intentions. Also, a mission may fail if a ship of a certain government scans your cargo and discovers you are carrying contraband.

SCAN_OUTFITS 

This ship scanned the given ship's outfits. (If it turns out the outfits include something illegal, this may result in a fine or an outright attack on the ship that was scanned.)

PROVOKE 

This ship damaged the given ship while not currently being an enemy of that ship's government; this will result in temporary animosities between the two governments. If a ship is "forbearing," it can only be "provoked" if its shields are below 90%. Some governments are provoked by starting a scan.

DISABLE 

This ship disabled the given ship. This will have a permanent effect on your reputation with the given government. This event is generated when a ship takes damage that switches it to being disabled.

BOARD 

This ship boarded the given ship. This may either be an attempt to render assistance, or an attempt to capture the ship.

CAPTURE 

This ship captured the given ship.

DESTROY 

This ship destroyed the given ship. If your projectiles hit a ship that is already exploding, that does not generate a "destroy" event; this is only for the one projectile that caused the explosion.

ATROCITY 

This is a crime that is so bad that it not only has a negative effect on your reputation, but entirely wipes out any positive reputation you had with the given government, first.

JUMP 

This ship just jumped into a different system.

ENCOUNTER 

This ship just met another ship for the first time
