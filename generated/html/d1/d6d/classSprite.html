Detailed Description

Class representing a drawable sprite. A sprite can have multiple frames, for animation. Each frame is stored in a separate OpenGL texture object. This may not be as efficient as sprite sheets, but with modern graphics cards it will not matter much and it makes working with the graphics a lot simpler.
