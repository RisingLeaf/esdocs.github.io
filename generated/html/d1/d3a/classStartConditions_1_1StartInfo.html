Detailed Description

Information to be shown to the player depending on current StartState. Information which is a child of the root node gets loaded into the UNLOCKED StartState. Information under "on (display | reveal)" nodes gets loaded into their respective VISIBLE or REVEALED states. The HIDDEN state has no information to display.
