◆ IsAlreadyOwned()

bool ShopPanel::IsAlreadyOwned ( ) const
protectedvirtual

Helper function for UI buttons to determine if the selected item is already owned. Affects if "Install" is shown for already owned items or if "Buy" is shown for items not yet owned.

If we are buying into cargo, then items in cargo don't count as already owned, but they count as "already installed" in cargo.
