Detailed Description

Class representing the outline of an object, with functions for checking if a line segment intersects that object or if a point is within a certain distance. The outline is represented in polygonal form, which allows intersection tests to be done much more efficiently than if we were testing individual pixels in the image itself.

Check if this mask intersects the given line segment (from sA to vA). If it does, return the fraction of the way along the segment where the intersection occurs.
