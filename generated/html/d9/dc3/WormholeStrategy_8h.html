enum class WormholeStrategy : int_fast8_t

Strategies that a DistanceMap can use to determine which wormholes to make use of when plotting a course to a destination, if any.

Enumerator
NONE 

Disallow use of any wormholes.

ONLY_UNRESTRICTED 

Disallow use of wormholes which the player cannot access, such as in the case of a wormhole that requires an attribute to use.

ALL 

Allow use of all wormholes.
