54// Class holding information about a "player" - their name, their finances, what
55// ship(s) they own and with what outfits, what systems they have visited, etc.
56// This class also handles saving the player's info to disk so it can be read
57// back in exactly the same state later. This includes what changes the player
58// has made to the universe, what jobs are being offered to them right now,
59// and what their current travel plan is, if any.
60class PlayerInfo {
61public:
62 struct FleetBalance {
63 int64_t maintenanceCosts = 0;
64 int64_t assetsReturns = 0;
65 };
